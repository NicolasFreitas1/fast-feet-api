FastFeet API

OpenAPI docs for the courier management API. The canonical contract is available in /docs/openapi.json.

Main demo credentials after running the seed: 11111111111 / 123456 (admin) and 22222222222 / 123456 (deliveryman).

Key flows

  • Admin login and entity management
  • Order marked as waiting by admin
  • Pickup, delivery with multipart proof, and return by deliveryman
  • Nearby orders and my deliveries for deliverymen

Important routes

  • POST /auth/login
  • PATCH /orders/:id/waiting
  • PATCH /orders/:id/pick-up
  • PATCH /orders/:id/deliver
  • PATCH /orders/:id/return

Multipart delivery example

PASSWORD=123456
TOKEN=<deliveryman JWT>
curl -X PATCH http://localhost:3333/orders/:id/deliver \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@proof.jpg"
